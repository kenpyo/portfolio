点字画像認識AI (AWS)

1. プロジェクトの背景と動機

視覚障がい者が利用する点字を、カメラ画像からリアルタイムで解析・テキスト変換することを目指しました。点字特有の「凹凸による影」や「色のなさ」という課題に対し、クラウドAIを用いたアプローチを試みました。

2. 使用したAWSサービス・技術

  • Amazon Rekognition: カスタムラベルを用いた画像認識モデルの学習・推論
  • Amazon SageMaker: モデルの構築・学習環境の管理
  • Amazon S3: 学習データおよび推論画像の管理
  • Python (Pillow): 学習用データセット（点字画像）の自動生成

3. 開発のプロセスと工夫点

データセットの自作 (Python)

日本語の点字画像データセットが不足していたため、PythonのPillowライブラリを用いて「あ」〜「ん」、濁音、句読点、数字の画像を各100枚ずつ自動生成し、学習データとして活用しました。

二段階の学習アプローチ

「一文字認識」と「文章内からの文字分割」の二段階でモデルを学習。一文字認識においては信頼度90%以上の良好な結果を得ることができました。

4. 成果と今後の課題

一文字認識モデルは完成し、精度の高い識別が可能となりました。一方で、文章全体からの文字分割や、実際の写真からの認識には、より多様な学習データ（実写データ）の拡充が必要であることを確認しました。
