点字画像認識AI

カメラ画像から点字を解析し、テキストへ変換するシステム。Amazon Rekognitionを活用し、点字のデジタルパターンを識別するモデルを構築しました。

※現在、AWSアカウントの期限満了に伴いソースコードおよび環境の閲覧が不可となっているため、開発当時の資料と推論結果を基に構成しています。

使用技術

  • AI / Model: Amazon Rekognition (Custom Labels)
  • Library / API: Python, Pillow (学習データ自動生成)
  • Logic / Tools: AWS SageMaker Ground Truth (アノテーション), S3

開発のプロセスと工夫

1. 日本語点字データセットの完全自作

学習に必要な点字画像が不足していたため、Python(Pillow)で正確なデジタルパターン画像を自動生成。学習データを自ら用意しました。

2. SageMakerを用いた効率的なアノテーション

生成画像に対し、SageMaker Ground Truthでバウンディングボックスを付与。文字認識の基礎となる「点字の範囲特定」に特化した学習プロセスを構築しました。

3. クラウドネイティブな開発フローの経験

S3でのデータ管理からRekognitionでの学習・推論まで、AWSのマネージドサービスを組み合わせた迅速なシステム構築を実践しました。
