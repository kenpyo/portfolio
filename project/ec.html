Webショッピングサイト

PHPとMySQLを用いた動的なECサイト。ユーザー側の会員登録やログイン買い物機能を実装しました。

使用技術

  • AI / Model: なし
  • Library / API: PHP , MySQL
  • Logic / Tools: ロリポップ！

開発のプロセスと工夫

1. データベース設計とCRUD処理の実装

商品・ユーザー・カートの各テーブルを設計。PHPのPDOを用いて、情報の登録・取得・更新・削除を確実に行えるバックエンドを構築しました。

2. セッションによる認証と状態維持

セッション管理機能を活用し、ログイン状態やカート内容を保持。ユーザーごとにパーソナライズされた購入体験を実装しました。

3. 外部サーバー（ロリポップ！）への公開

ローカル環境での開発後、ロリポップサーバーへデプロイ。FTP転送や本番環境でのDB連携設定など、公開までの一連のフローを完遂しました。

実際のサーバー上で動作するデモサイトを公開しています。
