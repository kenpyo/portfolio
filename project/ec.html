Webショッピングサイト

PHPとMySQLを用いた動的なECサイト。ユーザー側の買い物機能から、管理者側の在庫管理までを一貫して実装しました。

使用技術

  • AI / Model: なし
  • Library / API: PHP (PDO), MySQL (MariaDB)
  • Logic / Tools: ロリポップ！(サーバー), セッション管理, SQLによるデータ操作

開発のプロセスと工夫

1. データベース設計とCRUD処理の実装

商品・ユーザー・カートの各テーブルを適切に設計。PHPのPDOを用いて、情報の登録・取得・更新・削除を確実に行えるバックエンドを構築しました。

2. セッションによる認証と状態維持

セッション管理機能を活用し、ログイン状態やカート内容を保持。ユーザーごとにパーソナライズされた購入体験を実装しました。

3. 外部サーバー（ロリポップ！）への公開

ローカル環境での開発後、ロリポップサーバーへデプロイ。FTP転送や本番環境でのDB連携設定など、公開までの一連のフローを完遂しました。

評価と今後の展望

現在の到達点

Webアプリケーションの基本構造を習得。外部サーバーで公開・稼働させる実務に近い開発フローを経験しました。
